# Automation QA Exercises

This project demonstrates automated testing of backend APIs and JSON data using Node.js and Jest.


## Features
- Automates tests for REST APIs
- JSON data validation
- Example of request/response verification
- Reuseable test structures

## Getting Started
1. Clone the repo
2. Run 'npm install'
3. Run tests with 'npm test'

## Technologies
- Node.js
- Axois
- Jest
